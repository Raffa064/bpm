BPM (Bash Package Manager)

bpm <command> (arguments...)

Commands:
>help<    - Show this dialog
>version< - Display bpm version
>init<    - Creates a new bpm project
>package< - Open package.sh with your default text editor from >$EDITOR<
>install< - Install project dependencies
>list<    - List installed packages
>locator< - Update locator index for package
